# WebAPIドキュメント

日付: 2016-11-13

## ステータス

採用

## 決定事項

Springfoxを利用して、Swagger形式で運用する。

## コンテキスト

- WebAPIを公開するにあたり簡単にドキュメントを作成したい

## 決定理由

Swaggerは標準化団体が発足していて将来的にメインストリームとなると予想した。  
SpringfoxはSpring Bootのアプリケーションから自動的に作られるため、  
Spring Bootで作成する今回のアプリーケーションと相性が良いと考えた。
